# fzsm · 角色市场封面浏览

**线上：https://fzsm.vercel.app/**

## 功能

- 浏览器直连官方 `role_market.php` 列表
- **仅封面浏览**（搜索 / 排序 / 翻页）
- 无登录 / 无注册 / 无详情 / 无下载
- 用户自行设置起始页和结束页，点击「创建索引」后在浏览器本地重建
- 新建索引会先清空旧数据，完整重建时可移除已经下架的角色
- 以图搜图使用 aHash、dHash 与亮度轮廓特征进行本地比对
- 支持一次上传多张图片；缩略图显示在搜索区下方，点击可查看该图片对应的识图结果

## 结构

```
public/          静态前端
api/health.js    健康检查
api/config.js    公开配置
lib/sm.js                公开配置
vercel.json
```

## 部署

```bash
npx vercel --prod
```

## 本地手动索引

- 页面只读取当前浏览器 IndexedDB 中的本地索引，不请求云端索引。
- 设置「从第几页」和「到第几页」后点击「创建索引」。
- 创建前会清空旧索引，再逐页抓取角色封面、提取特征并保存到当前浏览器。
- 可随时取消；取消后保留本次已经完成的页面数据。
- 以图搜图仅比对本地索引，清除浏览器站点数据会同时清除索引。
- 部署不需要 Blob、Sharp、Cron 或任何环境变量。
